# Network Flexibility Design under Endogenous Uncertainty

This repository contains the code and instances accompanying the paper:

**Network Flexibility Design under Endogenous Uncertainty**  
Authors: Soroush Fatemi-Anaraki, Martin Grunow, Giovanni Pantuso, Stefan Weltge

The code implements the computational framework used in the paper. Given a set of plants and products, it computes a flexibility design that maximizes expected operational profit under supply and demand uncertainty. The model accounts for the fact that the chosen flexibility design affects not only the feasible production network, but also the probability distributions of uncertain supply and demand. In addition, several classes of valid inequalities are implemented to improve computational performance.

## Usage

The implementation is written in Python and requires the Gurobi solver to solve the underlying optimization problems.
